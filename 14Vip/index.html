会员权益须知
购物须知

1. 为维护您的权益，付款时请主动出示会员卡，以便享受到折扣优惠及积分。付款后请保留销售凭单及机打收银条，并核实是否刷会员卡。

2. 退货时请按北京君太百货相关规定办理。办理退货应扣除（退还）相应积分、礼金、赠品，如积分、礼金不足，应以同等价值现金补足。

积分说明

1. 办理会员卡前的消费金额不作积分。

2. 仅现金、信用卡、支票付款可享积分，如遇特别促销活动，积分规则以店内明示为准。

3. 不同会员卡积分不可累计使用；同一笔消费所得积分无法累积至不同会员卡。

4. 会员卡内积分一年一清（详见积分效期说明表格），逾期作废。

积分兑换

1. 卡内普通积分累计达1000分以上方可兑换(多倍积分除外)。

2. 积分兑换时须出示会员卡、办卡有效证件、款台机打收银条。

3. 礼金使用效期为兑换日起7天内有效，逾期作废。

4. 使用礼金消费，不再参加店内任何活动积分，不开具发票。

5. 多倍积分兑券比例及礼金使用规定依现场公告为准。

会员卡补办及更换

1. 会员卡为具会员资格之记名卡片，请妥善保管，不得外借。

2. 如不慎遗失、毁损，请会员本人携带有效证件（身份证、军官证、护照）及 10元工本费至卡务中心办理补卡手续。

3. 如会员卡遗失，有相关损失由会员本人承担。

4. 新办卡需10元工本费，退卡恕不退费。

会员权益

1. 会员所提供信息须真实有效，如有变更请及时更新，以免影响您的权益。

2. 特别活动须凭短信或邮件获得参加资格者，如未及时更新视为自动放弃。

3. 请依循《北京君太百货会员条款》使用规则。如有违反或有非正常目的消费，北京君太百货有权取消其会员资格。

4. 《北京君太百货会员条款》内容不定期更新，会员权益权限以店内明示为准。
